# Security Policy

## Supported Versions

The following versions of YourNotes are currently supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.2.x   | :white_check_mark: |
| 1.1.x   | :white_check_mark: |
| 1.0.x   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in YourNotes, we encourage you to report it responsibly. Please follow these steps:

1. **Do not** publicly disclose the vulnerability until it has been addressed.
2. Open a [new issue](https://github.com/Bilal140202/YourNotes/issues/new) with the label `security` and include as much detail as possible about the vulnerability.
3. Alternatively, you can contact the maintainer directly via [GitHub](https://github.com/Bilal140202).

We will acknowledge receipt of your report within 48 hours and aim to provide a resolution or timeline within 7 days. Once the vulnerability is resolved, we will publicly credit your discovery (unless you prefer to remain anonymous).
